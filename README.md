# Counter

Counter allows incrementing, decrementing or resetting the value.

## Installation

Clone repository.

```bash
git clone https://github.com/tiagors09/counter.git
```

Open project dir.

```bash
cd counter
```

Use the package manager [npm](https://www.npmjs.com/) to install counter dependencies.

```bash
npm install
npm run dev
```
